# TimeKeeper
A Python process's running time tracker

## Current Features
- [x] Dectect running processes
- [x] Track usage Duration
- [x] Store data in SQLite

## Quick Start
1. Clone this repo
2. Install dependecies: `pip install -r requirements.txt`
3. Run :  `python src/monitor.py`
